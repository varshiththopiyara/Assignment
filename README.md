### Product Listing Application

* Developed the **Product Display** page with a primary focus on creating a clean, responsive, and user-friendly UI.
* Implemented **pagination**, displaying **8 products per page** for better navigation and performance.
* Added **skeleton loading** to improve the user experience while product data is being fetched.
* Created a **Product Details** page (`/product/:id`) to display detailed information about each selected product.
* Included **Wishlist** and **Cart** icons on every product card. Currently, these buttons display an alert message and serve as placeholders for future functionality.
* Implemented a fully functional **search feature**, allowing users to quickly find products by name.
* Designed visually appealing and responsive **product cards** to provide an engaging browsing experience across different screen sizes.
